# CM111: Introduction to Programming with Java

Introduction to Programming with Java is an open source Java textbook licensed
under CC-BY 4.0.   

This book is a fork of "Java, Java, Java", released under the same license, and customized for use in CM111 at Washburn University with integration of other materials from [https://github.com/Washburn-CIS/CM111-Intro-to-Java](CM111-Intro-to-Java) and [CS1](https://github.com/jmorwick/cs1-java-book).
The fork was made on December 1th, 2025. More information on the original book, "Java, Java, Java" can be found here: (https://github.com/ram8647/javajavajava)[Java, Java, Java source].
